### USEAGE

#### single post
```bash
python jsl.py post single POST_ID
```

#### single user
```bash
python jsl.py user single --linkname=USERNAME
```

#### multiple posts or users
```bash
python jsl.py post multi --op=START_POST_ID --ed=END_POST_ID
```
```bash
python jsl.py user multi --op=START_USER_ID
```
